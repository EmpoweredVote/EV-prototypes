Complete Field Reference

Every field available from BallotReady GraphQL API. Fields marked NEW are fetched but not currently stored in our database.

OfficeHolder (top-level)
idStringGraphQL relay ID
databaseIdIntStored as external_id
isAppointedBooleanAvailable, not stored
isCurrentBooleanAlways true (filter)
isVacantBooleanAvailable, not stored
startAtStringStored as valid_from
endAtStringStored as valid_to
officeTitleStringStored as office_title
totalYearsInOffice NEWIntNot stored
Person
person.firstNameStringStored
person.middleNameStringFirst letter stored as middle_initial
person.lastNameStringStored
person.nicknameStringStored as preferred_name
person.suffixStringStored as name_suffix
person.fullNameStringReconstructed from parts
person.bioText NEWStringNot stored
person.bioguideId NEWStringNot stored (congress.gov link key)
person.slug NEWStringNot stored
person.images[] NEWArrayNot stored (we use photo_origin_url from Cicero only)
person.contacts[] NEWArraySeparate from officeholder contacts; not stored
person.degrees[] NEWArrayNot stored (degree, major, school, gradYear)
person.experiences[] NEWArrayNot stored (title, organization, type, start, end)
person.urls[]ArrayStored as urls[]
Position
position.nameStringStored as district_label + chamber_name
position.description NEWStringNot stored
position.levelStringMapped to district_type
position.appointedBooleanUsed in mapping, not stored directly
position.judicialBooleanUsed in mapping, not stored directly
position.tier NEWIntNot stored (1=federal, 2=state, 3=county, 4=local)
position.seats NEWIntNot stored
Other
parties[].shortName NEWStringNot stored (R, D, NP, etc.)
contacts[].fax NEWStringNot stored
contacts[].type NEWStringNot stored (central, office, primary, other)
addresses[].type NEWStringNot stored
urls[].typeStringUsed to classify identifiers (twitter, facebook, etc.) but not stored
